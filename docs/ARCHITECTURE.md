# 系統架構

## 資料流

```
┌──────────────┐     ┌─────────────────────────────────────┐
│  使用者輸入   │────▶│         Django Application          │
│ (追溯碼/名稱) │     │                                     │
└──────────────┘     │  ┌───────────┐  ┌──────────────────┐ │
                     │  │  views.py │──│ agent_service.py │ │
                     │  └───────────┘  └───────┬──────────┘ │
                     │                         │            │
                     └─────────────────────────┼────────────┘
                                               │
                    ┌──────────────────────────┼──────────┐
                    │                          │          │
                    ▼                          ▼          ▼
           ┌────────────────┐       ┌──────────────────┐
           │  TAFT API      │       │  FDA 本地資料庫   │
           │ (產銷履歷)      │       │ (食品業者登錄)    │
           │ data.gov.tw    │       │  PostgreSQL      │
           └────────┬───────┘       └────────┬─────────┘
                    │                        │
                    └──────────┬─────────────┘
                               ▼
                    ┌──────────────────┐
                     │   Gemini API     │
                     │ (資料整理+回答)   │
                    └────────┬─────────┘
                             │
                             ▼
                    ┌──────────────────┐
                    │  HTMX 前端渲染   │
                    │ (partials 更新)  │
                    └──────────────────┘
```

## 模組職責

### 1. 資料層

| 模組 | 類型 | 說明 |
|---|---|---|
| `services/taft_service.py` | API 串接 | 呼叫農業部產銷履歷 API |
| `services/fda_service.py` | 資料庫查詢 | 查本地食品業者登錄資料 |
| `management/commands/sync_fda_data.py` | ETL | 定期從 CSV 匯入食品業者資料 |

### 2. Agent 層

| 模組 | 說明 |
|---|---|
| `services/agent_service.py` | 合併兩個資料來源，送 Gemini API 產出自然語言回答 |

### 3. 表現層

| 模組 | 說明 |
|---|---|
| `views.py` | 接收查詢請求，呼叫 agent_service |
| `templates/` | HTMX 片段模板，無整頁刷新 |

## 注意事項

- Agent Service 是純資料整理者，不做知識推斷
- TAFT API 為即時查詢，FDA 為本地定期同步（可能有延遲）
- 所有外部 API 失敗時優雅降級（回傳 None/[]，不讓 view 炸掉）
