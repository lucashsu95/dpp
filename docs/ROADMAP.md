# 已知風險與限制

| 風險 | 說明 | 對策 |
|---|---|---|
| 資料造假 | 上游廠商申報不實，DPP 無法保證真實 | Agent 定位為「偵測異常」而非「保證真實」 |
| LLM 幻覺 | Agent 自行補充未有資料 | Prompt 嚴格限縮 + 顯示原始資料來源 |
| 食藥署無即時 API | 無法秒級同步業者登錄狀態 | 定期下載資料集，標註資料更新時間 |
| 平台寫入權限 | 無法直接寫入官方 DPP 平台 | 產出符合 DPP 格式的報告，由人工或企業上傳 |
